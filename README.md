# Deskripsi Aplikasi

Cafe Bar Way merupakan aplikasi katalog menu makanan dan minuman cafe dengan tampilan modern bertema coklat coffee shop.

Aplikasi memiliki:

- halaman login
- halaman daftar menu cafe
- sistem route protection sederhana
- desain responsive menggunakan Flexbox

---

# Fitur Aplikasi

## 🔐 Login Screen

- Tampilan login modern
- Tema warna cafe coklat
- Input email & password
- Tombol login menggunakan Expo Router

---

## Menu Cafe

Daftar menu:

- ☕ Cappuccino Latte
- 🥐 Croissant Butter
- 🍔 Burger Beef
- 🍕 Pizza Cheese
- 🍩 Chocolate Donut

---

## Detail Pesanan

Saat menu dipilih:

- nama menu tampil
- harga tampil
- status pesanan tampil

---

## Logout

- Menghapus token login
- Redirect kembali ke halaman login
